Potato Tomato
Two-Phase AI Pipeline
Camera Input
USB action camera · $80–150 hardware target · 0.8–1.0 m/s conveyor belt · continuous frame capture via threading + queues
Phase 1
Detection
Viola-Jones Haar Cascade on Sobel-preprocessed frames (10× downscaled)
IoU tracking prevents re-classification across frames
↓ fallback
MOG2 background subtraction + contour detection
Phase 2
Classification
Roboflow Inference API — crop-specific models: potato-detection-3et6q v11 tomato-detection-wb9kx v2
↓ local fallback (potato only)
Route 2A
SIFT + SVM
Dry Rot, Gangrene
Route 2B
Otsu → CNN (TFLite INT8)
Blights, scabs, surface
Grade Assignment
Grade result queued to actuator signal for physical sort
Phase 3 — Grading & Action
Keep
Grade A — full market value
Healthy, no surface defects detected
Secondary
Grade B/C — processing or hold
Sprouted, Common Scab <10% surface area, unripe tomato — animal feed or processing stream
Reject
Grade D/F — compost or destroy
Late Blight, Gangrene, Soft Rot, Dry Rot, severe fungal infection, Common Scab >10% surface
Review
Confidence <60% — manual inspection
Triggered when model certainty falls below threshold regardless of predicted class
